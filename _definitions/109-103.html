--- identity: DEF109 post: POST103 title: Nondeterministic Finite Automaton with ε-transitions synonyms: - Nondeterministic Finite State Machine - Nondeterministic Finite Acceptor ---

A nondeterministic finite automaton (ε-NFA) $$M$$ is a 5-tuple $$(Q, Σ, δ, q_0, F)$$, where

  1. $$Q$$ is a finite set called the states,
  2. $$Σ$$ is a finite set called the alphabet,
  3. $$δ : Q × Σ_ε → P(Q)$$ is the transition function and $$Σ_ε = Σ ∪ \{ε\}$$
  4. $$q_0 ∈ Q$$ is the start state, and
  5. $$F ⊆ Q$$ is the set of accept states.
