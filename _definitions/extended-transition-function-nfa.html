Let $$M = (Q, Σ, δ, q_0, F)$$ be a nondeterministic finite automaton. We define the extended transition function

$$$δ^* : Q × Σ^* → P(Q)$$$

as follows:

  1. For every $$q ∈ Q$$, $$δ^∗(q, ε) = \{q\}$$
  2. For every $$q ∈ Q$$, every $$y ∈ Σ^*$$, and every $$a ∈ Σ$$,

$$$δ^∗(q, ya ) = \bigcup \{δ(p,a) \mid p ∈ δ^∗(q, y)\}$$$
